Who We Are

ElectroHub is a student-led electronics website project designed to demonstrate web development skills using HTML and CSS. We showcase products, multimedia content, and professional website design suitable for academic submission.

Our Mission

To provide a professional, user-friendly platform demonstrating electronics products and web design skills. This project aims to combine practical coding skills with professional UI/UX design principles.

Our Vision

To inspire students to create professional-level websites using basic web technologies and to showcase creativity in design and layout.

Our Values

Professionalism, clarity, accessibility, and creativity are central to this project. We strive to make every page user-friendly and visually appealing for both academic and practical purposes.
